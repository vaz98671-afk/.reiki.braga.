# Como Publicar o seu Site no GitHub Pages

Siga estes passos para colocar o seu site online de forma gratuita, usando o GitHub Pages.

## Passo 1: Criar uma Conta no GitHub

Se ainda não tiver uma conta, crie uma em [https://github.com/join](https://github.com/join).

## Passo 2: Criar um Novo Repositório

1.  Vá a [https://github.com/new](https://github.com/new).
2.  **Nome do Repositório:** Escolha um nome para o seu projeto (ex: `reiki-clientes`).
3.  **Descrição (Opcional):** Adicione uma breve descrição do seu site.
4.  **Público/Privado:** Selecione **Público**. O GitHub Pages para repositórios privados é um recurso pago.
5.  Clique em **"Create repository"**.

## Passo 3: Carregar os Ficheiros do Site

Na página do seu novo repositório, siga estes passos:

1.  Clique em **"Add file"** e depois em **"Upload files"**.
2.  Arraste todos os ficheiros do seu site (`index.html`, `styles.css`, `contacts.js`, etc.) para a área de upload.
3.  No final da página, clique em **"Commit changes"**.

## Passo 4: Ativar o GitHub Pages

1.  Na página do seu repositório, clique em **"Settings"** (Definições).
2.  No menu à esquerda, clique em **"Pages"**.
3.  Na secção **"Branch"**, selecione o ramo **`main`** (ou `master`) e clique em **"Save"**.

## Passo 5: Aceder ao seu Site

Aguarde alguns minutos. O GitHub irá publicar o seu site e mostrar-lhe-á o link no topo da página de "Settings" > "Pages".

O seu site estará disponível no endereço:
`https://[seu-nome-de-utilizador].github.io/[nome-do-repositorio]`
